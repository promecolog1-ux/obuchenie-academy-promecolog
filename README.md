# Академия Промэколог

Обновлённый лендинг образовательного направления [Юридического центра промышленной экологии (ЮЦПЭ)](https://promecolog.ru). Сайт продаёт практическое обучение в сфере экологической безопасности — не дистанционные курсы ради документов, а программы с кейсами, тренажёрами проверки и образовательным порталом.

## Стек

- HTML5, CSS3, JavaScript (без фреймворков и сборщиков)
- Шрифты [Montserrat](https://fonts.google.com/specimen/Montserrat) (заголовки) и [Manrope](https://fonts.google.com/specimen/Manrope) (текст) — Google Fonts
- Официальный логотип ЮЦПЭ: `assets/logo.svg`

## Структура проекта

```
Обучениеэкологов.рф/
├── index.html          # Главная страница (лендинг)
├── styles.css          # Стили
├── script.js           # Интерактив: меню, FAQ, форма
├── assets/
│   └── logo.svg        # Логотип ЮЦПЭ (обрезанный viewBox)
└── legal/
    ├── privacy.html    # Политика конфиденциальности
    ├── consent.html    # Согласие на обработку ПДн
    ├── terms.html      # Пользовательское соглашение
    ├── education.html  # Сведения об образовательной деятельности
    └── license.html    # Лицензия
```

## Запуск локально

Сайт статический — достаточно открыть `index.html` в браузере или поднять локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

После запуска откройте `http://localhost:8080`.

## Программы обучения

1. **Для руководителей** — обеспечение экологической безопасности руководителями общехозяйственных систем управления (72 ак. часа)
2. **Для специалистов** — обеспечение экологической безопасности специалистами общехозяйственных систем управления (72 ак. часа)
3. **По отходам** — обеспечение экологической безопасности при работах в области обращения с отходами I–IV классов (38 ак. часов)

## Форма заявки

Форма в блоке «Получить программу и консультацию» (`#contact`):

- Поля: имя, компания, телефон, email, комментарий
- Кнопки «Получить программу» и «Обсудить корпоративное обучение»
- Сейчас заявки сохраняются в `localStorage` браузера (ключ `academy_leads`) — для демонстрации

**Для продакшена** нужно подключить отправку на email или CRM (Битрикс24 / amoCRM), как указано в ТЗ.

## Логотип

Используется официальный SVG ЮЦПЭ. В файле `assets/logo.svg` скорректирован `viewBox`, чтобы убрать лишние поля холста и показывать логотип в нужном масштабе. Пропорции и элементы логотипа не изменялись.
